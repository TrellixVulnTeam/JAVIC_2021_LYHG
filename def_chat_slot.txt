-chatbot dialog
  -welcome message(or block, intent) : 첫 인사 메뉴등..
  -fallback message : 양해를 구하는 메세지

  -domain : 해당 서비스의 전문 용어
  -context : 대화 문맥

  -entity : 도메인 대화모델에서 사용자와 시스템이 주고 받는 개념 및 속성
    -class : 범주, 먼저 클래스의 특정 개념을 정의하고 그 클래스의 세부 속성을 slot으로 정의
    -slot : 해당 클래스가 가지고 있는 여러가지 속성 모든 슬롯을 채울 때까지 대화 진행

  -NER : 개체명 인식 입력된 개체명을 추출하여 인명 지명 기관명 시간등으로 분류하여 인식
